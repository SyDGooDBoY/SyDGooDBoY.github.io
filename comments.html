Comments

This is my first time designing a website, although there are so many things I have to learn, I finished it after searching about 50 html/css tutorial websites.

Technical

The component of this website is 4 pages and 1 CSS style sheet.

  1. index.html

  2. past.html

  3. future.html

  4. comments.html

  5. websystems.css

Structure

This website has been separated by “div” into 3 parts:

  1. The navbar and title on the top of the page.(div id="label")

  2. The contents of the page. (div id="PagenameContents")

  3. The footer( div id="footer")

Selector

In these 4 HTML pages, I used the “id” selector to group all the common parts such as the title on the top of the page and the navigation bar. The reason I use the “id” selector is these parts are the same on every page.For the part of the contents, I used the “class” selector to make them can be edited more accurate.

CSS

In the CSS stylesheet, I layout the common parts firstly to show a basic interface and then I can layout the contents part will be much easier.

Aesthetics

The main aesthetic colour of this page:

  1. Black (#000000)

  2. Green (#10eb77)

  3. Magenta (#fc0ff5)

The inspiration for these colours is the terminal console. The input text colour of most terminal is green and black (dark mode), green already become a symbol of computer language and black can make the text looks more mysterious. The inspiration for magenta is from some cyberpunk movies and games, this colour can give the users a sense of science fiction, it can combine with the terminal colour perfectly.

Interaction

The hyperlink in this site has been designed with 4 types. The normal type of the hyperlink is the same colour as the contents, when the mouse hovering on the tag, the background will become green and the text will become magenta, once the mouse clicks, the background colour and text colour will flip-flap, after the link has been visited, the colour will become magenta forever. The hyperlink on the navbar has no underscore but the hyperlink in the contents has because I want to make the hyperlink can catch the eye to let the user view it.

Accessibility

The responsivbe web is a necessary function of a website, all the device has different size. In this website, I used the code (meta name="viewport" content="width=device-width, initial-scale=1.0") to make it can be adaptive zoom with the browser.
